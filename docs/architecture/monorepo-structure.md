# Monorepo Structure

## Папки верхнего уровня

- `apps/` — пользовательские и worker приложения.
- `services/` — backend сервисы (HTTP API на FastAPI).
- `packages/` — shared/domain/contracts пакеты.
- `infra/` — инфраструктурные артефакты.
- `data/` — локальные данные и артефакты.
- `docs/` — архитектура и этапы реализации.

## API-стандарт

Все HTTP сервисы в монорепозитории строятся на:

- FastAPI для роутинга и валидации;
- uvicorn для запуска ASGI приложения.

Минимальный контракт каждого сервиса:

- `GET /health`
