# TTT Monorepo

Монорепозиторий для сервиса мониторинга билетов с обязательной двухканальной проверкой цен:

- `Channel A`: быстрый источник (агрегатор/API/кэшированный парсер)
- `Channel B`: прямой сайт авиакомпании
